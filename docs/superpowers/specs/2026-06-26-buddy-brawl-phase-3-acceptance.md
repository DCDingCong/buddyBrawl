## 首版可用路径

1. 用户打开小程序。
2. 点击“开始游戏”。
3. 授权手机号。
4. 服务端创建或加载账号，并返回下一步动作。
5. 完整账号进入“我的熊猫”主页。
6. 用户查看熊猫体型、资源、挂机收益和下一步建议。
7. 用户进入冒险页领取收益或挑战关卡。
8. 用户进入装备页用金币强化装备。
9. 用户进入竞技页挑战对手、查看战报和排行榜。

## 已知上线限制

- 当前后端手机号登录支持开发 mock 凭证和本地闭环验证；正式开放用户前需要接入微信官方手机号凭证换取接口，并配置小程序 AppID/AppSecret。
- 小程序默认 API 地址是 `http://127.0.0.1:3000`；正式开放用户前需要部署 HTTPS 后端，并在微信公众平台配置合法请求域名。
- 当前验收在本地 Docker 和 HTTP 接口完成；微信开发者工具中的真机/模拟器手工验收仍需要用 `apps/miniprogram` 项目目录执行。
- 角色初始化页已存在，但当前首版后端会在手机号登录时直接创建完整角色，因此正常新用户路径会直接进入主页。
- 首版不包含支付、广告、商城、正式外观库存、好友挑战和赛季系统。
